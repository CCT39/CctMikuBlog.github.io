感謝您造訪「CCT的咪苦筆記」網站（以下簡稱「本網站」）。我們重視您的隱私權，特制定本隱私權政策，說明本網站如何蒐集、使用及保護您的資訊。

一、資料蒐集與使用#

1.1 Google Analytics 4 (GA4)#

本網站使用 Google Analytics 4（GA4）來分析網站流量與使用者行為。Google 會透過 Cookie 及類似技術蒐集您的匿名使用數據，包括但不限於：

  • 瀏覽的頁面與內容
  • 停留時間與互動行為
  • 裝置類型、作業系統與瀏覽器資訊
  • 大致的地理位置（國家/地區層級）
  • 來源網站與搜尋關鍵字

這些資訊僅用於改善網站體驗、分析流量趨勢與優化內容，不包含可直接識別您個人身分的資訊（如姓名、電子郵件地址等）。

1.2 Google 信號（Google Signals）#

本網站未啟用 Google Analytics 的「Google 信號」廣告功能。因此，我們不會蒐集您的 Google 帳號相關聯的人口統計資訊或興趣資料。

1.3 Cookie 使用說明#

本網站透過 Google Analytics 使用 Cookie 來儲存與讀取資料。Cookie 是儲存在您裝置上的小型文字檔案，用於識別您的瀏覽器並記錄您的偏好設定。您可以透過瀏覽器設定來管理或刪除 Cookie，但這可能會影響網站的部分功能。

二、資料共享與第三方服務#

2.1 Google 服務#

本網站使用的 Google Analytics 服務，其資料處理與隱私保護遵循 Google 隱私權政策 及 Google Analytics 服務條款。

2.2 GitHub Pages 託管服務#

本網站託管於 GitHub Pages。GitHub 可能會蒐集伺服器日誌數據（如 IP 地址、瀏覽器類型、參照頁面）以維護服務的安全性與穩定性。這些資料的處理方式遵循 GitHub 隱私權聲明。

三、您的權利與選擇#

3.1 拒絕追蹤（Opt-out）#

您有權選擇不被 Google Analytics 追蹤。以下是幾種拒絕追蹤的方式：

  • 安裝瀏覽器外掛： 您可以安裝 Google Analytics 不顯示瀏覽器外掛，以防止您的資料被 Google Analytics 蒐集。
  • 管理 Cookie 設定： 您可以透過瀏覽器設定來封鎖或刪除 Cookie。
  • 關閉廣告個人化： 您可以透過 Google 廣告設定 來管理廣告個人化的相關設定。

3.2 瀏覽器的「Do Not Track」功能#

目前本網站尚未支援瀏覽器的「Do Not Track」（DNT）訊號，但您仍可透過上述方式來拒絕追蹤。

四、資料安全#

我們重視您的資料安全，並已採取合理的技術與管理措施來保護您的資訊。然而，網路傳輸無法保證絕對安全，我們無法完全排除資料外洩的風險。

五、兒童隱私#

本網站不針對 13 歲以下兒童設計，亦不會刻意蒐集兒童的個人資料。如果我們發現無意中蒐集了兒童的資料，將立即刪除相關資訊。

六、隱私權政策的變更#

我們可能會不定期更新本隱私權政策。任何重大變更將於本頁面公告，並更新「最後更新日期」。建議您定期查閱本政策以了解最新資訊。
